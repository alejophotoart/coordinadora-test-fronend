# Frontend Coordinadora Envios

Este proyecto fue generado usando [Angular CLI](https://github.com/angular/angular-cli) version 19.2.4.

## Servidor de desarrollo

Comando para arrancar servidor de localhost:
Abrir consola:

```bash
ng serve
```

Una vez el servidor este corriendo abra en su navegador este link `http://localhost:4200/`. La aplicacion automaticamente
cargara todos los archivos en ella

## Instalacion de dependencias

Para este proyecto se instalaron algunas dependencias utiles para su desarrollo

Abrir consola y ejecutar

```bash
ng i
```
o

```bash
ng install
```

Esto fue todo por el frontend
